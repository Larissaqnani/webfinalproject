It is used to embed an image in a web page.

<ul> e <li>

Lists allow to group a set of related items in lists

  • <ul> define an unordered list
  • <li> define each item

<ol> e <li>

Lists allow to group a set of related items in lists.

  1. <ol> define an ordered
  2. <ul> define each item

<div>

It is used as a container for other HTML elements. It takes all available width, and comes with line breaks before and after.

<span>

Define a small text area, where it can be configured individually.

<form>

Define a form to input data that can be process in a datacenter.

<input>

An input field that allows the user to enter data.

<button>

Create a clickable button.

<table>, <tr>, <th>, <td>

Create a table, rows, header and data cell

<strong> e <em>

<blockquote>

Defines a blockquote section.

<abbr>

Define an abreviation with its explanation.

<address>

Define address informations about the author or sender.

<del> e <ins>

  • <del> define a delete text but keep it on the code
  • <ins> define a new text part, that was updated.

<code>

Define and configure a block code that is in the text.

<cite>

Define the title of a work, such as a book or a publication.

Define a caption for an element <figure>.

Defines a specific date or time, which can be read, formatted and manipulated by machines.

  • <sup> define superscript text
