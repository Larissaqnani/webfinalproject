font-family

Defines the font family to be used for the text in an element.

font-size

Defines the font size to be used for the text in an element.

font-style

Sets the font style to italic, normal or oblique.

font-weight

Defines the thickness of the text, making it thinner or thicker.

color

Defines the color of the text in an element.

text-decoration

Sets text decoration, such as underline, overline, dashed...

background-color

Defines the background color of the element

margin

Defines the external margin of the element. Might be used to create space between the elements.

padding

Create space around the element's content

border

Allows to specify the style, width, and color of an element's border.

text-align

Defines the horizontal text align inside the element.

line-height

Sets the line height in a text block, controlling the space between lines of text.

display

Defines how an element will be dispayed: block, inline, ou none.

position

Defines the method of positioning an element in relation to its container.

float

Defines whether an element should float left, right.

opacity

Defines the opacity of an element, where 0 is completely transparent and 1 is completely opaque.

animation

Define a custom animation for an element, with full control over duration, direction and repetition.

width and height

Define the width and height of the element

Box Model (margin, border, padding)

The box model defines the space around and inside an element: margin, border and padding.

cursor

Defines the type of cursor to be displayed when the mouse hovers over an element.

box-shadow

Adds and configure a shadow around an element, creating an illusion of depth.
